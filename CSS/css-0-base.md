## 介绍

CSS(Cascading Style Sheets) 译为层叠样式表

## 结构

```css
p {                                              /* 选择器, 选择一个或多个元素 */
    color: red;                                  /* 属性名: 属性名字;  设定属性值 */
    width: 500px;
}

#html-id {                                       /* id 选择器 */

}

.html-clss {                                     /* 类选择器 */

}

a:hover {                                        /* 伪类选择器, 特定状态的元素 */

}
```

## 盒模型

|:-:|:-|:-|:-|
|描述|内边距(边框和内容距离)|边框(边框厚度)|外边距(边框与外部距离)|
